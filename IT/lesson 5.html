CSS специфичность (приоритеты правил)

Подсчитывается три типа параметров:
  • A - #id
  • B - классы или псевдоклассы
  • C - теги элементов (div, ::before)
Записывается десятичное число из цифр ABC Формально число не в 10-чной системе, а в системе с достаточно большим основанием AN^3 + BN^3 + CN (где N достаточно велико). Чем число больше, тем правило приоритетней.

св-во transition: all 2s;
